Guía para la Degradación del suelo
Google Earth Engine

La Wildlife Conservation Society (WCS) ha desarrollado una herramienta en Google Earth Engine para generar mosaicos satelitales y detectar cambios en el uso del suelo mediante algoritmos avanzados. Esta herramienta está diseñada específicamente para la región del Beni.

¿Qué necesitas?

  1. Registro en Google Earth Engine
    Ingresa a https://earthengine.google.com y solicita acceso con tu cuenta Gmail , recibiras un correo de confirmación
  2. Ingreso del Script
    Una vez que se tenga acceso a una cuenta de google, abre el siguiente enlace para ejecutar el script.

Vista previa del script

La interfaz muestra el área de interés con los resultados de degradación para la gestión 2024 fue determinada a partir de un script que integra tres variables: cobertura de uso del suelo, cuerpos de agua y riesgo de incendio, permitiendo identificar las zonas con mayor afectación.
